# To do list - Список дел

Приложение позволяет записать список необходимых дел.

## Описание

**Создано с помощью REACT, классовый компонент**

+ Добавляй с помощью `Enter` или кнопки `Добавить`
+ Вычёркивай сделанное
+ Удаляй весь список
